# Бэкенд проекта детекции HumanValues в текстах

Данный проект располагает кодом, хранилищем и интерфейсом доступа к предобученным моделям проекта. 

---

## Установка проекта

1. Скачайте проект на ваше. Перейдите при помощи команды cd в диррикторию проекта
2. Убедитесь, что находитесь в корне проекта. Затем выховите команду компиляции Docker Image
`./scripts/build.sh`
Он может собираться достаточно долго, нужно быть готовым. Последующие установки будут быстрее из-за кэширования билда
3. Для поднятия контейнера запустите ./scripts/run.sh


Чтобы быстро убивать и заново поднимать контейнер используйте команду:

```docker stop $(docker ps -a -q) && docker rm $(docker ps -a -q); ./scripts/build.sh;  ./scripts/run.sh```

--- 

## Интерфейс

Вам доступны два эндпоинта в py модуле `./src/api.py`: 

1. `/predict/human-values` – принимает текст и кодификатор модели
2. `available_models` – возвращает все досутпные модели как их кодификаторы 

---

## Обращение к бэкенду

Если вы правильно поставили контейнер, то перейдя по адресу 

`http://localhost:8000/docs` вы сможете увидеть весь список эндпоинтов
